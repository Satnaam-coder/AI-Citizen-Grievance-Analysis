# AI-Citizen-Grievance-Analysis
AI Driven Citizen Grievance and Sentiment Analysis System using NLP and Machine Learning.

This project analyzes citizen complaints data using data science techniques such that data cleaning , exploratory data analysis, and machine learning to identify complaint patterns.

# AI Driven Citizen Grievance Analysis

This project analyzes citizen complaints data using Data Science and Machine Learning techniques. The goal of this project is to understand complaint patterns, identify common issues, and analyze complaint trends.

# Dataset

The dataset used in this project is the 311 Service Requests dataset(data available on Kaggle). It contains citizen complaints related to public services.

# Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Machine Learning Model Building
6. Model Evaluation

# Exploratory Data Analysis

EDA was performed to understand complaint patterns and distributions.

Some of the analysis performed:

- Complaint category distribution
- Location wise complaints
- Complaint length analysis
- Visualization using bar charts and plots

# Objective

The objective of this project is to analyze citizen grievances and apply machine learning techniques to classify complaints and understand public service issues.

# Results

- Logistic Regression Accuracy: ~87%
- Naive Bayes Accuracy: ~87%
- Decision Tree Accuracy: ~87%
- Random Forest Accuracy: ~87%

All models performed similarly. Naive Bayes was selected as the final model.
